# `1-bit-wonder`

Full-stack web developer by trade, low-level tinkerer by inclination. The two have more in common than they look — both are about stripping away layers until there's nothing left to hide behind, whether that's a static HTML page or a 1-bit CPU.

### By night — built for the love of it

- **[cpu-1bw14500b](https://github.com/1-bit-wonder/cpu-1bw14500b)** — A cycle-accurate Rust simulator of the Motorola MC14500B, a genuinely 1-bit processor from 1977: a one-bit ALU, 16 opcodes, no general-purpose registers. Verified against the chip's published instruction timing. This is where the name comes from.
- **[noctalia-nixos-tracker](https://github.com/1-bit-wonder/noctalia-nixos-tracker)** — A Luau desktop plugin that notifies me the instant a nixpkgs PR lands on a target branch. The upstream tracker exposes no API, so it parses server-rendered HTML; it runs from the read-only Nix store by keeping all mutable state in `$XDG_STATE_HOME`, and dedupes notifications across restarts.
- **[noctix-os](https://github.com/1-bit-wonder/noctix-os)** — My entire machine as one file: a reproducible Hyprland desktop and a nix-darwin Mac built from a single cross-platform Nix flake, with VM and ISO outputs so a clean install is one `nix build` away.

### By day

Full-stack web since 2017. I started out in PHP/Laravel, but these days I reach for the simplest thing that does the job: Astro, TypeScript, and a lot of plain HTML and CSS — adding a database only when the problem genuinely needs one. Less framework, less to break, faster pages.
